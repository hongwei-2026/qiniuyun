# VoiceCanvas 测试报告

> 测试：于鸿明 · 开发：于鸿伟  
> 日期：2026-06-14

## 测试环境

- Windows 11，Chrome / Edge
- 后端 `uvicorn app.main:app --port 8000`
- 前端 `npm run dev` → http://localhost:5173
- 识别：讯飞极速转写 + 浏览器识别对照

| 场景 | 语音指令示例 | 结果 |

| 矢量绘图 | 画红色圆形、在左上角画蓝色正方形 | 通过 |
| 样式变换 | 改成蓝色、放大一倍、撤销 | 通过 |

## 已知限制

- 讯飞 OST 转写有 2～5 秒延迟，属 API 特性
- 需配置 `backend/.env` 后 AI 生图/漫画/意图解析才可用
- Demo 视频见 README B 站链接
