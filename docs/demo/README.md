## 赛题提交方式

按 [PROJECT_GUIDE.md](../../PROJECT_GUIDE.md) 第七节：

1. 将 `voicecanvas-demo.mp4` 上传至 **B 站** 或 **网盘**（需可公开访问）
2. 把播放链接写入仓库根目录 [README.md](../../README.md) 的「Demo 视频」一节
3. 推送代码仓库时**只需推送链接**，无需推送 mp4 本体
