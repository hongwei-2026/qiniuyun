# 语音指令速查（评委参考）

> 完整交互式手册：运行项目后说 **「指令手册」** 打开 UI 弹窗。  
> 源码定义见 `frontend/src/data/commandManual.ts`。

## 系统与聆听

| 指令示例 | 说明 |
|----------|------|
| 开始聆听 / 停止聆听 / 重新聆听 | 控制麦克风 |
| 切换讯飞识别 / 切换浏览器识别 | ASR 引擎切换 |
| 停止生成 | 中断 AI 生图或漫画绘制 |
| 指令手册 / 查看指令 / 帮助 | 打开或听取指令说明 |
| 关闭指令手册 | 关闭手册弹窗 |

## 模式切换

| 指令示例 |
|----------|
| 切换到自由画布 / AI 创作 / 九宫格 / 漫画创作 / 3D |
| 打开资产管理 |

## 自由画布

- **绘图**：画红色圆形、画矩形、画五角星、画波浪线、画流程图、画火柴人
- **样式**：改成蓝色、线宽 5、实心圆、空心
- **变换**：放大一倍、向左移、旋转、居中
- **画布**：撤销、重做、清空、保存图片、放大、适应窗口

## AI 创作

- 生成赛博朋克城市、横版生成森林、重新生成、生成变体
- 切换豆包 / 切换 MiniMax
- 保存图片、占满画布

## 九宫格

- 新建九宫格、切成九宫格、重绘第 3 格、向上扩格
- 统一风格、导出瓦片集、地图初始化

## 漫画创作

- **项目**：新建漫画项目、打开默认漫画、切换到漫画二
- **设定**：设定风格为…、设定故事背景为…
- **角色/剧本**：画角色立绘小明、创作第一集剧本
- **生成**：生成第一集漫画、生成第一集和第二集漫画
- **重绘**：重绘第一集第五页和第三页
- **导出**：导出 PDF

## 3D

- 把当前图生成 3D 模型

## 复杂指令

- 画圆然后画矩形然后保存（多步自动执行）
- 口语化长句由 DeepSeek 拆解为多个工具顺序执行
