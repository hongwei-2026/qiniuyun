## 1. 计划支持的指令能力

### 1.1 基础绘图（自由画布）

| 类别 | 计划指令 |
|------|----------|
| 图形 | 线段、圆/椭圆、矩形、三角形、多边形、五角星、箭头、波浪线、流程图、火柴人 |
| 样式 | 颜色（红/蓝/绿等口语）、线宽、实心/空心、透明度 |
| 变换 | 移动、缩放、旋转、翻转、复制、删除、置顶/置底 |
| 画布 | 撤销、重做、放大/缩小、适应窗口、清空、保存 PNG |

### 1.2 AI 绘画

- 文生图（MiniMax / 豆包 Seedream 可切换）
- 重新生成、生成变体、横版/竖版比例
- 局部重绘（选中区域）
- 图生图参考（当前画布作为参考）

### 1.3 九宫格素材

- 新建九宫格、切成九宫格、选中/重绘指定格
- 向上/下/左/右扩格、移格、批量铺格
- 统一风格、导出瓦片集
- 工作流宏：地图初始化（一键创建 + 切格 + 命名）

### 1.4 漫画创作

- 多漫画项目管理（新建、切换、打开默认漫画）
- 全局风格与故事背景设定
- 角色立绘生成与资产管理
- 分集剧本创作（LLM）
- 分镜页批量生图、多集并发生成
- 指定页/多页重绘、中文对白 Canvas 叠加
- PDF 导出

### 1.5 3D

- 当前 2D 图 → 豆包 Seed3D 图生模型
- Three.js 预览、OBJ 下载

### 1.6 语音与系统

- 连续聆听 / 停止 / 重新聆听
- 浏览器 Web Speech API / 讯飞 OST 高精度 ASR 切换
- 自适应断句：短句快速执行、句中停顿延长等待、完整句再解析
- 本地快路径：简单矢量指令不经 LLM，降低延迟
- 复杂句 DeepSeek 拆解为多 Tool 顺序执行 + 分步 TTS 播报
- 口语归一化与同义词容错（如「九官格」→「九宫格」）
- 内置指令手册（语音或 UI 打开）

## 2. 最终实现的指令能力

### 2.1 核心绘图与画布 ✅

| 能力 | 语音示例 |
|------|----------|
| 基础图形 | 画一个红色圆形、画矩形、画五角星、画波浪线 |
| 样式 | 改成蓝色、线宽 5、实心圆 |
| 变换 | 放大一倍、向左移、旋转、居中 |
| 画布管理 | 撤销、重做、放大、适应窗口、清空、保存图片 |

### 2.2 AI 绘画 ✅

| 能力 | 语音示例 |
|------|----------|
| 文生图 | 生成赛博朋克城市、横版生成森林 |
| 重试与变体 | 重新生成、生成变体 |
| 提供商切换 | 切换豆包、切换 MiniMax |
| 局部重绘 | 重绘选中区域为雪地 |

### 2.3 九宫格 ✅

| 能力 | 语音示例 |
|------|----------|
| 创建与切格 | 新建九宫格、切成九宫格 |
| 格级编辑 | 重绘第 3 格、向上扩格 |
| 批量与导出 | 统一风格、导出瓦片集、地图初始化 |

### 2.4 漫画创作 ✅

| 能力 | 语音示例 |
|------|----------|
| 项目管理 | 新建漫画项目、打开默认漫画、切换到漫画二 |
| 设定 | 设定风格为日系漫画、设定故事背景为… |
| 角色与剧本 | 画角色立绘小明、创作第一集剧本 |
| 生成 | 生成第一集漫画、生成第一集和第二集漫画 |
| 重绘 | 重绘第一集第五页和第三页 |
| 导出 | 导出 PDF |

### 2.5 3D ✅

| 能力 | 语音示例 |
|------|----------|
| 图生 3D | 把当前图生成 3D 模型 |

### 2.6 语音系统 ✅

| 能力 | 说明 |
|------|------|
| 双 ASR | 浏览器识别 + 讯飞 OST，语音切换 |
| 本地快路径 | 简单绘图指令 <100ms 级响应 |
| 自适应断句 | 短确认语（如「欧克」）快速执行；长句可停顿思考 |
| 多步执行 | 「画圆然后画矩形然后保存」自动分步 |
| 指令手册 | 「指令手册」「查看指令」「帮助」 |
| 浏览器 ASR 断线重连 | Edge/Chrome 长时间聆听自动恢复 |

### 2.7 性能与容错（实现要点）

- **指令理解**：本地正则 + 漫画/九宫格专用解析器优先，复杂句走 DeepSeek Tool Calling
- **容错**：中文数字页码/集数解析、项目名模糊匹配、同义口语归一化
- **响应**：简单指令本地直执行；生图/漫画异步带进度 TTS 播报

完整指令速查见 **[VOICE_COMMANDS.md](./VOICE_COMMANDS.md)**；运行时可说「指令手册」查看 UI 版。

## 3. 未完成部分及原因

| 功能 | 状态 | 原因 |
|------|------|------|
| 实时讯飞 WebSocket 流式 ASR | 未实现 | 当前采用 OST 异步转写，准确率高但延迟高于流式；流式需单独产品开通与前端改造 |
| 多画布持久化项目管理 | 部分 | 漫画/角色数据存 IndexedDB；自由画布仅 PNG 导出，未做云端同步 |
| 自由手绘笔触路径 | 未实现 | 纯语音难以精确控制笔触轨迹，与题目「结构化指令」方向不一致 |
| 图层树语音精细管理 | 部分 | 支持置顶/置底与选中，未做完整图层列表面板（避免鼠标依赖） |
| 离线/无 API Key 全功能 | 部分 | 矢量绘图与浏览器 ASR 可离线；AI 生图/漫画/意图解析依赖外部 API |
| 移动端适配 | 未实现 | 开发周期聚焦桌面 Chrome/Edge 麦克风体验 |
