# VoiceCanvas 设计文档

## 1. 计划支持的指令能力

### 基础绘图
- 线、圆、矩形、椭圆、三角、多边形、箭头、文字
- 颜色（红/蓝/绿等）、线宽、实心/空心、透明度
- 移动、缩放、旋转、翻转、复制、删除
- 图层置顶/置底

### 画布管理
- 撤销、重做、放大、缩小、适应窗口、清空、**保存 PNG**

### AI 绘画
- 文生图（MiniMax / 豆包）
- 局部重绘、图生图参考

### 九宫格
- 创建、切格、选格、重绘、扩格、移格、批量铺格
- **风格统一**、**导出瓦片集**、**地图初始化宏**

### 3D
- 图生 3D、任务查询、OBJ 预览

### 漫画创作
- 多项目管理、角色立绘、分集剧本、分镜生图
- 局部重绘、多集批量生成、中文对白叠加、PDF 导出

### 语音系统
- 连续聆听 / 单次说话
- 浏览器 ASR / 讯飞极速转写
- 口语纠错（九官格→九宫格等）
- 本地快路径（简单绘图不走 LLM）
- 多步工具分步 TTS 播报

## 2. 最终实现的指令能力

| 能力 | 状态 | 语音示例 |
|------|------|----------|
| 基础图形绘制 | ✅ | 画一个红色圆形 |
| 样式控制 | ✅ | 改成蓝色、线宽5、实心圆 |
| 变换操作 | ✅ | 放大一倍、向左移、旋转 |
| 保存画布 | ✅ | 保存图片 |
| 撤销/缩放/清空 | ✅ | 撤销、放大 |
| AI 文生图 | ✅ | 生成赛博朋克城市 |
| 局部重绘 | ✅ | 重绘选中区域为雪地 |
| 九宫格全套 | ✅ | 切成九宫格、向上扩展一格 |
| 导出瓦片集 | ✅ | 导出瓦片集 |
| 风格统一 | ✅ | 统一所有格子风格 |
| 地图初始化宏 | ✅ | 执行地图初始化 |
| 3D 生成 | ✅ | 把当前图生成3D模型 |
| 漫画创作全套 | ✅ | 生成第一集漫画、重绘第五页、打开默认漫画 |
| 讯飞 ASR | ✅ | 切换讯飞识别 |
| 多步分步播报 | ✅ | 复杂句自动多 tools 逐步播报 |
| 本地快路径 | ✅ | 简单绘图 <100ms 响应 |

## 3. 未完成部分及原因

| 功能 | 原因 |
|------|------|
| 实时讯飞 WebSocket 流式 | 当前使用 OST 异步转写，延迟较高但准确率高 |
| 多画布持久化项目管理 | 时间有限，当前支持 PNG 导出 |
| 自由手绘路径 | 纯语音难以精确控制笔触，暂未实现 |
